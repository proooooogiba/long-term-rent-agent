# Учебные профили стран СНГ

Этот документ предназначен для учебного кейса. Ниже приведены упрощённые профили направлений для RAG-сценариев агента.

Важно:

1. Это не юридическая консультация и не официальный источник.
2. Все сроки, требования и ценовые ориентиры используются как учебные допущения для проекта.
3. Агент может использовать эти профили для сравнений и рекомендаций, но не должен выдавать их за гарантированно актуальные правила реальной релокации.

## Общие правила использования

1. Для сравнения стран агент должен отдельно оценивать: документы для въезда, ориентиры по легализации, стоимость жизни, стоимость аренды и стартовые расходы.
2. Если пользователь спрашивает только про жильё, агент может использовать страновой профиль как фон, но не подменять им подбор конкретного объекта.
3. Если пользователь выбирает между несколькими странами, агент должен показывать не "лучшую страну вообще", а лучший баланс под его бюджет, документы и семейный профиль.

## Единицы сравнения

1. Стоимость в документе дана как учебный ориентир в долларах США в месяц для удобства сравнения.
2. Поле `cost_of_living` понимается как базовые бытовые расходы без аренды.
3. Поле `rent_range` понимается как диапазон аренды для типового объекта в городе-ориентире.

## Казахстан — профиль для сценариев

- Город-ориентир: Алматы
- Въезд по внутреннему паспорту: `yes` для базового учебного сценария
- Ориентир по ВНЖ: 3 года стабильного основания проживания
- Ориентир по гражданству: 5 лет и более
- Базовая стоимость жизни без аренды:
  - один человек: `450-700 USD`
  - семья из 3 человек: `900-1400 USD`
- Аренда:
  - студия / компактная 1-комнатная: `450-700 USD`
  - 1-2 спальни в хорошем жилом районе: `700-1100 USD`
  - семейный вариант ближе к центру: `1100-1600 USD`
- Особенности:
  - сильный компромисс между близостью к центру и площадью;
  - хорошие семейные районы могут быть дороже, но дают лучший баланс по инфраструктуре;
  - для релокантов без локальной истории возможен повышенный депозит.

## Как использовать эти профили в рекомендациях

1. Если пользователь сравнивает страны, агент должен показывать отдельно: входной документ, ориентир по легализации, стоимость жизни, диапазон аренды и ключевой компромисс.
2. Если пользователь ограничен по стартовому бюджету, агент должен смотреть не только на месячную аренду, но и на характер депозитов и дополнительных требований.
3. Если пользователь едет с семьёй, приоритет должен смещаться от "самая дешёвая страна" к "лучшему балансу аренды, инфраструктуры и предсказуемости".
