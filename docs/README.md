# Документация проекта

Этот каталог хранит подробные материалы по relocation agent. Главный `README` оставлен коротким для быстрого входа, а здесь лежат архитектура, оценка, demo-материалы и ограничения.

## Основные документы

- [project_overview.md](project_overview.md) — расширенный обзор проекта: почему это агент, роли, архитектура, данные, метрики, ограничения и roadmap.
- [demo_scenarios.md](demo_scenarios.md) — готовые demo-сценарии, runbook и скриншоты.
- [evaluation_framework.md](evaluation_framework.md) — как устроена оценка агента, quality gate и benchmark-подход.
- [production_operating_model.md](production_operating_model.md) — operating notes по auth, data freshness, observability и target production contour.
- [project_criteria_gap_analysis.md](project_criteria_gap_analysis.md) — разбор покрытия критериев и оставшихся gap'ов.
- [external_mcp_provider_example.md](external_mcp_provider_example.md) — пример подключения внешнего MCP-провайдера объявлений.
- [known_issues_root_causes.md](known_issues_root_causes.md) — известные проблемы и разбор причин.
